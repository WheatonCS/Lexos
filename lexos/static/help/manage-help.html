The Manage Tool

Manage is the tool you use to perform various types of "housekeeping" on documents in your Lexos workspace. In addition to documents derived from files you have uploaded, Manage will also list documents created by other tools, such as segments produced by the Cutter tool.

Usage

  • Purposes

    • Activate and de-activate documents in your workspace (Lexos tools only operate on your active documents). Delete unwanted documents from your workspace. Re-name documents in your workspace. Add class labels to your documents in your workspace (e.g. apply an AUTHOR class label to all documents with the same author).

  • Columns in the Document Table

    • Documents in your workspace are listed in the form of a table. The left-most column indicates what documents are active.

    The "Document" column lists the filename without the filename extension. The original filenames are listed in the "Source" column. If you use Lexos tools to create new documents based on your uploaded files (e.g., if you segment a document), the original filename will be displayed in the "Source" column, and a new name will be generated for the "Document" column. Document names can be changed using the context menu, as described below.

    By default, documents created by file upload or a Lexos tool do not have an associated class, so the "Class" column is empty.

    The "Excerpt" column shows a preview of the beginning of each document.

    Rows in the table can be sorted by column information by clicking on the column header (see the arrow in the column header to indicate the sorting order, e.g., increasing or decreasing order).

Options

  • Activating and De-activating Documents

    • By default, all documents are activated when they are uploaded. In subsequent analyses, only active documents are used. Return here to ensure all desired documents are active.

    The following methods can be used to manage the active state of documents:

    • Single-Click: This will activate the document selected and toggle the state of the row clicked. If the document is active, it will be deactivated.
    • Drag-Click: Clicking on a row with the mouse button held down will activate (or de-activate if the documents are already active) all rows between the row clicked and the row the mouse cursor is over when the mouse button is released.
    • Right-Click: This will open the context menu. See Using the Context Menu below.

  • Deleting Documents

    • Deleting individual documents from the workspace is probably achieved most easily achieved using the Context Menu as described below. However, if you wish to delete all your documents and re-start your workflow, use the "Reset" button in the top right to delete all documents and begin from the Upload page.

Using the Context Menu

Right-clicking on a table cell or row will open the context menu. It has the following options:

  • Preview Document

    • This will open a dialog containing the entire text of your document (without formatting or white spaces). Note that longer documents can take a while to load, so please be patient.

  • Edit Name

    • This function allows you to create a new name for the document in the row you have clicked. To change the name, enter your new name in the dialogue form field and click "Ok".

  • Edit Class

    • This function allows you to create a class label for the single document in the row you have clicked. Enter the label you wish to identify with the class in the dialogue form field and click "Ok". See the section on Classifying Documents below.

  • Delete

    • This function will delete the individual document in the row you have clicked.

  • Merge

    • This function will merge selected documents together. You can give the merged document a new name as you add it to the corpus list, while the un-merged documents will remain untouched. You can also add a milestone to mark in the text where one document ends and another begins.

  • Edit Active Classes

    • If you have multiple active documents, this option will allow you to apply a class label to all of them at once. Enter the label you wish to identify with the class in the dialogue form field and click "Ok". See the section onClassifying Documents below.

  • Delete Active

    • If you have multiple active documents, this option will allow you to delete them all at once.

  • Activate All and Deactivate All

    • Activate All will activate all uploaded documents. Deactivate all will deactivate all uploaded documents.

Classifying Documents

Document classes are groups of documents identified as belonging to the same category defined by some human-assigned criterion. For instance, a collection of novels might be separated into two classes based on whether they were published in Britain or the United States. Gender, genre, date of authorship, and authorship might be used to classify documents. Lexos' class labels allow you to assign classes to documents and sort by class in the Manage tool. At present, document classes are under-utilized elsewhere in the Lexos suite; however, they are an important part of the Topwords tool. In general, you should assign class labels in Manage before using Topwords.

Manage Features Outside this Page

By clicking the "Active Documents" text in the bottom right corner of any other page, you can open a pop-up window containing a smaller version of this page. From there, you can activate and deactivate documents, though you cannot use any features accessed with the right-click.

Note that the page will not update to reflect the changed state of your documents, but the document states displayed in the pop-up are correct and will be used when interacting with the current page.
